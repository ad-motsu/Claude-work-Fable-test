AWS Certified Solutions Architect - Associate (SAA-C03)

SAA-C03 学習テキスト（第1章・第2章）

クラウド実務経験1年程度のエンジニア向けに、AWS公式情報源のみを根拠として作成した学習テキストです。 本テキストは第1章・第2章のみを対象としています（第3章・第4章は範囲外です）。

目次

  • Domain 1 ・ 試験比重 30%

    第1章　セキュアなアーキテクチャの設計

    IAMとアクセス設計 / VPCセキュリティと脅威対策サービス / データ暗号化と鍵管理　＋　章末問題5問
  • Domain 2 ・ 試験比重 26%

    第2章　弾力性に優れたアーキテクチャの設計

    マルチAZ/マルチリージョン設計 / 疎結合アーキテクチャ / コンピュートとスケーリング / ストレージ・DBの可用性 / バックアップ・DR戦略　＋　章末問題5問

本テキストがカバーするのはDomain 1・Domain 2のみです。Domain 3（高性能アーキテクチャ）・Domain 4（コスト最適化アーキテクチャ）は本テキストのスコープ外です。

学習の進め方

  1. まず第1章のイントロで「責任共有モデル」を理解し、1.1（IAM）→1.2（ネットワーク/脅威対策）→1.3（データ保護）の順に読み進めてください。
  2. 各節の末尾には参照した公式情報源のURLを明記しています。理解が浅いと感じた用語は、必ずリンク先の一次情報を確認してください。
  3. 章末問題は、解答を自分で考えてから解説を読むようにしてください。誤答した問題は、該当する節に戻って該当箇所を読み直すことを推奨します。
  4. 第1章を終えたら第2章（マルチAZ/マルチリージョン設計から始まる回復性の設計）に進んでください。
  5. 本テキストはDomain 1・2のみを扱う教材です。試験全体（Domain 3・4を含む）の対策には、本テキストに加えてAWS公式試験ガイドや追加の学習教材を利用してください。

試験概要（AWS公式試験ガイド・公式認定ページで確認済み）

項目内容
出題数65問（スコア対象50問＋未スコア15問。未スコア問題は試験中に区別されない）
出題形式multiple choice（単一正解＋誤答3つ）／multiple response（2つ以上の正解を選択）
試験時間130分
合格スコア720（スケールスコア100〜1000のうち）
採点方式コンペンセイトリー方式（総合点で合否判定。セクションごとの合格基準はない）
受験料150 USD

ドメイン別の出題比重は次のとおりです。

ドメイン比重本テキストでの扱い
Domain 1: Design Secure Architectures30%第1章として収録
Domain 2: Design Resilient Architectures26%第2章として収録
Domain 3: Design High-Performing Architectures24%本テキストの範囲外
Domain 4: Design Cost-Optimized Architectures20%本テキストの範囲外
